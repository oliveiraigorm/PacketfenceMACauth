# PacketfenceMACauth

.csv sample file :

00:15:E9:2B:99:3C,default
00:00:00:00:99:32,default
00:15:E9:2B:99:34,professor
00:15:E9:00:99:3C,default
00:00:00:55:99:32,default
00:15:E9:11:99:34,professor
